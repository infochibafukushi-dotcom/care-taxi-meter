# 介護タクシーメーターアプリ 実機テストチェックリスト

## 目的

Phase1-A〜E 完了後の実機検証として、介護タクシー業務で日常運用できる品質になっているかを確認する。

対象端末：

* 主対象：OPPO Reno9 A
* 補助対象：Androidタブレット
* 補助対象：PCブラウザ

確認観点：

* 実機で画面が正常表示されるか
* ボタンが押しやすいか
* 状態遷移が想定通りか
* GPS距離計測が動作するか
* 低速時間加算が二重課金されないか
* 精算が正しくできるか
* 障害者割引が正しく反映されるか
* タクシー券が正しく反映されるか
* 複数決済が正しく反映されるか
* 領収書PDFが正しく出るか
* 利用明細書PDFが正しく出るか
* 売上分析に正しく反映されるか
* 監査ログが保存されるか
* 権限ごとの操作制限が正しく動くか
