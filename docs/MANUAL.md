# 介護タクシー専用メーター 現状機能マニュアル

この文書は、現在のアプリに実装されている機能を画面別に列挙し、日常操作の流れをまとめた利用マニュアルです。

## 1. 画面・URL一覧

| URL | 画面名 | 主な用途 |
| --- | --- | --- |
| `/` | TOP / ログイン | スタッフログイン、出勤・退勤、本日実績の確認、主要画面への移動 |
| `/case/start` | 案件開始 | 出勤中のスタッフ・車両を確認して案件を開始 |
| `/case` | 業務用メーター | 運行ステータス、GPS距離、料金、介助、実費、精算、案件保存、帳票発行 |
| `/cases` | 案件一覧 | 保存済み案件の一覧確認と詳細画面への移動 |
| `/cases/:caseRecordId` | 案件詳細 | 保存済み案件の明細確認、領収書PDF出力 |
| `/admin` | 管理センター | 会社・料金・帳票・スタッフ・店舗・車両・簡易売上・システム設定 |
| `/admin/analytics` | 売上分析 | 期間・スタッフ別の売上分析、CSV出力 |
| `/hq` | FC本部管理 | superAdmin向けの加盟店管理 |

## 2. TOP / ログイン画面（`/`）

### 2.1 ログイン

1. 会社ID、スタッフID、パスワードを入力します。
2. 「ログイン」を押します。
3. 認証に成功するとTOPダッシュボードへ移動し、勤務中のセッションが残っている場合は勤務中状態を復元します。

### 2.2 出勤

1. ログイン後、未出勤の場合は「出勤」を押します。
2. ブラウザの位置情報が利用できる場合、出勤時の緯度・経度・精度を保存します。
3. 出勤後は勤務時間が1秒ごとに更新されます。

### 2.3 TOPダッシュボード

TOPでは以下を確認できます。

- 会社、店舗、担当者
- 出勤時刻、勤務時間、出勤状態
- 本日件数、本日売上、本日走行距離、本日待機時間、本日付き添い時間
- 当月件数、当月売上、平均単価を含む退勤前サマリー

### 2.4 主要メニュー

出勤中は以下へ移動できます。

- 案件開始
- 案件一覧
- 管理センター
- FC本部管理（権限が superAdmin の場合に利用）

### 2.5 退勤

1. 「退勤」を押すと退勤サマリーを表示します。
2. 内容確認後に退勤を確定すると、退勤位置を取得して勤務セッションを終了します。

## 3. 案件開始画面（`/case/start`）

1. TOPで出勤済みの状態にします。
2. 案件開始画面で現在の店舗・スタッフを確認します。
3. 稼働中かつ同一会社・同一店舗の車両を選択します。
4. 「案件を開始」を押すと、選択車両付きで業務用メーターへ移動します。

## 4. 業務用メーター画面（`/case`）

### 4.1 案件車両選択

- TOPで出勤しているスタッフ名、店舗名を表示します。
- 車両は、出勤中の会社・店舗に紐づく「稼働中」「表示ON」の車両から選択します。
- 未出勤の場合、案件保存はできません。

### 4.2 運行ステータス

現在のステータスは以下です。

- 空車
- 待機中
- 院内付き添い中
- 走行中
- 精算前
- 案件終了

主な操作は以下です。

1. 「運行開始」で走行中にします。
2. 「待機」で待機中にします。
3. 「付き添い」で院内付き添い中にします。
4. 「空車」で空車に戻します。
5. 「精算・終了」で精算フローを開きます。

走行中はGPS取得が有効になり、空車または案件終了ではGPS取得を停止します。待機・付き添いは、それぞれ初回開始後から課金対象時間として扱います。

### 4.3 GPS・距離計算

- 走行中はブラウザ位置情報を5秒ごとに取得します。
- 時間距離併用運賃として、GPS速度が10km/h以下の区間は距離を加算せず低速時間を加算し、10km/hを超える区間のみ距離を加算します。
- 距離加算はGPS精度30m以内、前回GPS位置からの移動距離5m以上、かつ5秒で500m以上の異常移動ではない場合に限ります。
- GPSパネルでは、現在の緯度・経度、精度、速度、GPSログ件数、累計走行距離、エラーを確認できます。

### 4.4 住所取得

- 運行開始時に伺い先住所を取得します。
- 精算開始時に送り先住所を取得します。
- Google Maps JavaScript API / Geocoding API を利用します。
- 住所取得診断では、APIロード状態、Geocoder生成状態、geocode実行状態、レスポンス件数、formatted_address、エラーメッセージなどを確認できます。

### 4.5 料金表示

メーターには以下を表示します。

- 現在料金
- 次回加算額と次回加算までの残距離
- 基本運賃、待機料金、付き添い料金、介助料金、実費、合計の内訳
- 距離、運行時間、待機時間、付き添い時間

### 4.6 介助料金

1. 「介助」モーダルを開きます。
2. 管理センターで有効化されている介助項目を選択します。
3. 選択済みの介助項目はサマリーに表示され、合計料金へ加算されます。
4. テンキー入力で名称・金額を指定した追加介助料金を登録できます。

### 4.7 実費

1. 「実費」モーダルを開きます。
2. 管理センターで登録された実費プリセットを選択します。
3. テンキー入力で任意の名称・金額の実費を追加できます。
4. 追加済み実費はサマリーに表示され、合計料金へ加算されます。

### 4.8 一時料金設定

メーター画面の「設定」から、その案件画面上で参照している料金設定を確認・一時編集できます。

- 初乗り距離、初乗り料金
- 加算距離、加算料金
- 待機30分単位料金
- 付き添い30分単位料金
- 有効な介助料金の金額
- 実費ボタンの名称・金額

本格的な永続設定は管理センターの料金設定で行います。

### 4.9 精算・保存・帳票発行

1. 「精算・終了」を押します。
2. 支払方法を選択します。選択肢は「現金」「クレジット」「QR決済」「請求書」「その他」です。
3. 「保存」を押すとFirestoreへ案件を保存します。
4. 保存後、「レシート発行」または「領収書発行」を押します。
5. 発行完了後、「新しい案件を開始」で画面をリロードして次案件を開始します。

保存される主な内容は、案件番号、開始・終了・保存日時、距離、運転・待機・付き添い時間、勤務セッション、車両、料金内訳、支払方法、伺い先・送り先位置/住所、介助項目、実費です。

## 5. 案件一覧画面（`/cases`）

- Firestoreに保存された案件を一覧表示します。
- 案件番号、日時、担当者、車両、距離、合計金額、支払方法などを確認できます。
- 各案件から詳細画面へ移動できます。

## 6. 案件詳細画面（`/cases/:caseRecordId`）

案件詳細では以下を確認できます。

- 案件番号、利用日時、開始日時、終了日時
- 担当者、車両名、車両ナンバー、店舗、勤務ID
- 運転時間、待機時間、付き添い時間
- 伺い先住所、送り先住所
- 距離、基本運賃、待機料金、付き添い料金、介助料金、実費、合計金額
- 支払方法

領収書発行設定で、宛名、発行担当者、但し書きを入力してPDF出力できます。

## 7. 管理センター（`/admin`）

### 7.1 管理センター概要

管理センターには以下のカードがあります。

- スタッフ管理
- 車両管理
- 店舗管理
- 会社情報・帳票設定
- 料金設定
- 売上分析
- システム設定

画面上部では、保存済み案件数、総売上、出勤中スタッフ数などの管理サマリーを確認できます。

### 7.2 スタッフ管理

スタッフの追加・編集・保存ができます。

編集項目は以下です。

- 表示ON/OFF
- 表示順
- 会社ID
- 店舗
- スタッフ名
- ユーザーID
- パスワード
- role（superAdmin / owner / manager / driver）
- 電話番号
- メール
- 住所
- 免許番号
- 免許期限
- 事故歴
- メモ

### 7.3 車両管理

車両の追加・編集・保存ができます。

編集項目は以下です。

- 表示ON/OFF
- 表示順
- 会社ID
- 店舗
- 車両名
- ナンバー
- 状態（稼働中 / 整備中 / 休車 / 売却済）
- 燃料（未設定 / ガソリン / 軽油 / EV）
- 車両種別
- 車いす台数
- ストレッチャー対応
- 車検期限
- 保険期限
- メモ

### 7.4 店舗管理

- Phase1の最小構成として、初期店舗を登録できます。
- 店舗一覧で表示順、表示ON/OFF、会社ID、店舗ID、店舗名を確認できます。

### 7.5 会社情報・帳票設定

領収書・利用明細書・レシートに利用する会社情報と帳票設定を1つの画面で管理します。

会社情報:

- 法人名
- 屋号名
- 電話番号
- メールアドレス
- 郵便番号
- 住所

帳票設定:

- 発行担当者
- 領収書タイトル
- 利用明細書タイトル
- 適格請求書発行事業者登録番号
- 但し書きデフォルト

保存した会社情報・帳票設定は、領収書・利用明細書・レシートに自動反映されます。表示対象は屋号名、法人名、住所、電話番号、登録番号、発行担当者、但し書きです。

### 7.6 料金設定

以下をFirestoreへ保存できます。

- 初乗り距離
- 初乗り料金
- 加算距離
- 加算料金
- 待機30分単位料金
- 付き添い30分単位料金
- 介助項目の名称、金額、有効/無効、並び順
- 実費プリセットの名称、デフォルト金額

### 7.7 帳票出力への反映

登録番号が未設定の場合、PDFには登録番号を表示しません。

### 7.8 管理センター内の簡易売上分析

- 支払方法別の件数・売上を表示します。
- 月別売上、件数、平均単価を表示します。
- 「詳細な売上分析を開く」から売上分析画面へ移動できます。

### 7.9 システム設定

現状では以下の情報表示が中心です。

- データバックアップ: Phase1では新規実装なし
- バージョン情報: 管理センター Phase1
- 権限管理: スタッフ管理の role 設定を利用
- 今後の管理者向け設定の入口

## 8. 売上分析画面（`/admin/analytics`）

### 8.1 フィルター

- 開始年月、終了年月で分析期間を指定できます。
- スタッフを「全スタッフ」または個別スタッフから選択できます。

### 8.2 KPI・集計

以下の集計を表示します。

- 総売上
- 案件数
- 平均単価
- 総走行距離
- 基本運賃、待機料金、付き添い料金、介助料金、実費
- 支払方法分析
- 介助項目分析
- 実費分析
- 距離帯分析
- エリア別集計表
- スタッフ別集計
- 月別推移
- 月別一覧表
- 売上TOP10案件

### 8.3 CSV出力

「CSV出力」で、指定期間・指定スタッフの案件明細をCSVとしてダウンロードできます。

## 9. FC本部管理画面（`/hq`）

### 9.1 権限

- FC本部管理画面は superAdmin のみ利用できます。
- superAdmin以外は利用不可メッセージとTOPへ戻るリンクを表示します。

### 9.2 加盟店追加

以下を入力して加盟店を追加できます。

- 会社ID
- 加盟店名
- オーナー名
- 電話番号
- メール
- 並び順
- 住所
- メモ

### 9.3 加盟店一覧・詳細

加盟店一覧では以下を確認できます。

- 会社ID、加盟店名、状態
- 店舗数、スタッフ数、車両数
- 案件数、売上
- 詳細表示
- 稼働中加盟店の停止

詳細では、選択した加盟店の店舗数、スタッフ数、車両数、案件数、売上を確認できます。

## 10. 帳票機能

### 10.1 レシートPDF

- メーター画面の精算保存後に「レシート発行」から出力します。
- サーマルレシート向けの細長いPDFを生成します。
- 案件番号、利用日時、料金内訳、支払方法、登録番号、発行担当者などを印字します。

### 10.2 A4領収書PDF

- メーター画面の精算保存後、または案件詳細画面から出力します。
- 宛名、発行担当者、但し書きを指定できます。
- 案件番号、利用日時、距離、料金内訳、合計金額、支払方法、登録番号などを印字します。

## 11. データ保存先・外部サービス

- Firebase / Firestore: スタッフ、車両、店舗、勤務セッション、案件、メーター設定、会社情報を保存します。
- ブラウザ Geolocation API: 出勤・退勤位置、運行中GPS、住所取得用座標に利用します。
- Google Maps JavaScript API / Geocoding API: 緯度経度から住所を取得します。
- LocalStorage: 介助・実費のテンキー入力履歴を保存します。

## 12. 基本的な日次業務フロー

1. TOPで会社ID、スタッフID、パスワードを入力してログインします。
2. 「出勤」を押して勤務を開始します。
3. 「案件開始」へ進み、案件車両を選択します。
4. 業務用メーターで「運行開始」を押します。
5. 必要に応じて「待機」「付き添い」「介助」「実費」を入力します。
6. 目的地到着後、「精算・終了」を押します。
7. 支払方法を選択し、「保存」を押します。
8. レシートまたは領収書を発行します。
9. 「新しい案件を開始」で次案件へ進みます。
10. 1日の業務終了時にTOPへ戻り、「退勤」します。

## 13. 初期設定・運用前チェック

1. Firebase環境変数を設定します。
2. Google Maps APIキーを設定し、Maps JavaScript API と Geocoding API を有効にします。
3. 管理センターで初期店舗を登録します。
4. スタッフを登録し、ログイン用のユーザーID・パスワードを設定します。
5. 車両を登録し、状態を「稼働中」、表示をONにします。
6. 料金設定、介助項目、実費プリセットを確認・保存します。
7. 会社情報・帳票設定で法人名、屋号名、登録番号、発行担当者、但し書きを設定します。
8. 実機スマートフォンで位置情報許可、GPS距離、住所取得診断、PDF出力を確認します。
